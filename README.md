## Como rodar o projeto?

Para instalar o projeto, use:

```cmd
  yarn install
```

ou

```cmd
  npm install
```

E, depois, para inicializar o projeto, use:

```cmd
  yarn start
```

ou

```cmd
  npm start
```

Para testar sua aplicação, rode :

```cmd
  yarn teste
```

ou

```cmd
  npm test
```

E, por fim, para buildar sua aplicação para produção, use:

```cmd
  yarn build
```

ou

```cmd
  npm build
```
